    1. 前言

    入侵检测系统（intrusion detection system，简称“IDS”）是一种对网络传输进行即时监视，在发现可疑传输时发出警报或者采取主动反应措施的网络安全设备。它与其他网络安全设备的不同之处便在于，IDS是一种积极主动的安全防护技术。 IDS最早出现在1980年4月。 1980年代中期，IDS逐渐发展成为入侵检测专家系统（IDES）。 1990年，IDS分化为基于网络的IDS和基于主机的IDS。后又出现分布式IDS。目前，IDS发展迅速，已有人宣称IDS可以完全取代防火墙。

    第一部分我们要实现的功能是提取五元组, 既源IP地址，源端口，目的IP地址，目的端口，和传输层协议

    5. 准备解析数据包

    在解析数据包前，我们需要先知道数据包的结构，经查询，pcap的结构如下：

    |Global Header|Packet Header|Packet Data|Packet Header|Packet Data|…|

    我们之前获取到的header就是Packet Header，而data就在Packet Data中， Packet Data的结构如下:

    |Ethernet Header|IPv4 Header|Data Packet|…|

    Ethernet Header (14 Bytes)

    bytes Name
    6 目的 MAC 地址
    6 源 MAC 地址
    2 以太网类型

    IPv4 Header (20 Bytes)

    bits Name
    4 IP Version Number (IP版本号 (4))
    4 IHL (Internet Header Length 报头长度)
    8 Type of Service (服务类型)
    16 Total Length (IP包总长)
    16 Identification (标识符)
    4 Flags (标志)
    12 Fragment Offset (片偏移)
    8 Time to Live (生存时间)
    8 Protocol (协议)
    16 Header Checksum (头校验和)
    32 Source Address (源IP地址)
    32 Destination Address (目的IP地址)

    TCP Packet

    Protocol

    id Protocol Number
    1 ICMP
    2 IGMP
    6 TCP
    17 UDP
    88 IGRP
    89 OSPF

    Mac相关的东西我们暂时不关心，直接看IP相关的东西，这里我们需要找到几个ip的内容：ip版本，ip数据，从结构中可以看出IP版本在header中的前4bit， 所以可以直接取出来:

    ord(ip_data[0:1]) >> 4

    IP版本分IPv4和IPv6，之后可以分开处理。

    ip_data中的内容很多，如果简单地位移操作取出来的话太麻烦，所以我们可以使用struct库进行unpack

    Struct format

    Format C Type Python type Standard size Notes
    x pad byte no value
    c char string of length 1 1
    b signed char integer 1 (3)
    B unsigned char integer 1 (3)
    ? _Bool bool 1 (1)
    h short integer 2 (3)
    H unsigned short integer 2 (3)
    i int integer 4 (3)
    I unsigned int integer 4 (3)
    l long integer 4 (3)
    L unsigned long integer 4 (3)
    q long long integer 8 (2), (3)
    Q unsigned long long integer 8 (2), (3)
    f float float 4 (4)
    d double float 8 (4)
    s char[] string 1
    p char[] string
    P void * integer (5), (3)

    通过struct.unpack("!BBHHHBBH4s4s", ip_data[:20])可以取出ip_header, 我们可以再提取出src_ip和dst_ip，用socket.inet_ntoa()转换成我们能看的格式。

    之后我们先处理TCP包，从上图可以知道tcp包中的内容和长度，我们把需要用的port到flags(NS~FIN)之间的内容取出来就行了，使用如下语句：

    src_port, dst_port, _, _, doff_reserved, flags = struct.unpack(“!HHLLBB”, ip_data[iph_length:iph_length + 14])

    到这儿我们需要的五元组已经全部提取出来了。
